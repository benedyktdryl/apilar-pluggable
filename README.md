## Key Features

- Minimal API
- Allow to create plugins system
- Allow to share limited API
- Handle synchronous & asynchronous dependencies

## Install

Using NPM:

```bash
npm install --save-dev @apilar/pluggable
```

Using Yarn:

```bash
yarn add -D @apilar/pluggable
```
